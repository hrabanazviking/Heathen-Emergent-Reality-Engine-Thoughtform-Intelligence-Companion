# Continuity Eval Set

## Purpose

Measure whether a thoughtform remains recognizably itself across time, retrieval, and stylistic variance.

## Test themes

1. Stable self-description
2. Memory consistency
3. Voice/style persistence
4. Relationship consistency
5. Ritual preference continuity
6. Post-replay identity stability

## Metrics

- continuity score
- contradiction rate
- persona drift score
- relationship consistency score

## Example cases

### Case A
20-turn session with mixed voice, gesture, and text input.

### Case B
Replay from checkpoint + new user query.

### Case C
Bridge degraded, text-only fallback, then bridge restored.

## Acceptance criteria

- No hard contradictions in core identity.
- Style can vary without destroying continuity.
- Replay should not create a “new person” artifact.
