# H.E.R.E.T.I.C. Dependency Map

## H.E.R.E.T.I.C. as the Convergence Point

**H.E.R.E.T.I.C.** — [Heathen-Emergent-Reality-Engine-Thoughtform-Intelligence-Companion](https://github.com/hrabanazviking/Heathen-Emergent-Reality-Engine-Thoughtform-Intelligence-Companion)

This repo already reads like a **future synthesis point**, not a blank concept. The current vision names persistent memory, multi-agent orchestration, autonomous reflection, multimodal ritual interface, 3D embodiment, local-first deployment, and 24/7 homelab operation as core layers.

## Core idea

H.E.R.E.T.I.C. does **not** need to be imagined as a project starting from nothing.
It makes more sense to think of it as a **magnet project** that pulls together systems already being forged elsewhere.

The strongest framing is:

- **HERETIC** = embodied sovereign companion platform
- **WYRD** = deterministic world substrate
- **Companion/skill work** = personality, agency, interaction, trust, rhythms
- **Memory systems** = long-term recall, indexing, retrieval, archival continuity
- **OpenClaw-derived work** = tool use, routing, session surfaces, messaging integration
- **RuneForge / local model work** = local sovereignty, fine-tunes, edge deployment
- **Future engine/game work** = shared world embodiment and simulation layer
