## 3. CI/CD Pipeline Example
```yaml
name: model-release
on: [workflow_dispatch]
jobs:
  eval-and-release:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - run: pip install -e .[dev]
      - run: pytest -q
      - run: python scripts/run_eval_suite.py --profile release_candidate
      - run: python scripts/promote_model.py --if-gates-pass
```
