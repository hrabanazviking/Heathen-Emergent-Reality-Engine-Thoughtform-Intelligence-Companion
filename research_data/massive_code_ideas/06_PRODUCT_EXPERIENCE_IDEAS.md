# 06 — Product and Experience Code Ideas

## A. Conversation Design

### Idea A1: Presence Modes
- Modes:
  - practical guide
  - reflective companion
  - ritual storyteller
- Mode switch includes clear user-visible confirmation.

### Idea A2: Session Arc Planner
- Generate per-session arc:
  - opening recap
  - focus segment
  - closure + next-step carryover

### Idea A3: Ritual Interaction Blocks
- Encapsulate recurring experiences:
  - morning alignment
  - evening reflection
  - intention-setting

## B. Proactive System Features

### Idea B1: Notification Reasoning Engine
- Require justification tuple before outreach:
  - reason
  - relevance score
  - consent check
  - user fatigue estimate

### Idea B2: Adaptive Tone Tuner
- Tone model informed by:
  - user preference profile
  - recent sentiment trajectory
  - conversation context class

### Idea B3: Callback Memory Prompts
- Suggest callbacks only when confidence and user value exceed threshold.

## C. Multimodal Experience

### Idea C1: Voice State Controller
- Align TTS prosody to intent class and emotional resonance settings.

### Idea C2: Visual Symbol Composer
- Generate iconography or rune motifs tied to session themes.

### Idea C3: Ambient UX Hooks
- Optional background soundscape cues for reflective experiences.
